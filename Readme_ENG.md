1. PROJECT DESCRIPTION
This project consists of building an electronic Iambic Keyer with an integrated Morse code decoder, based on the ESP32‑C3 Super Mini microcontroller.

The device allows you to:

Transmit Morse code using an iambic paddle (two levers: dot and dash)

Automatically decode the transmitted Morse code

Display the decoded text on a 4‑line OLED screen

Adjust transmission speed (WPM)

Automatic spacing after 1 second pause

Automatic screen clear after completing 4 lines

Manual clear with a short button press
